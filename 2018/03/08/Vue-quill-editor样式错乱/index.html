2018年1月19日，项目使用
加入富文本编辑器（quillEditor），出现如下问题.

查询一圈，是因为这玩意儿升级了，css要单独去引用 

import 'quill/dist/quill.core.css'  
import 'quill/dist/quill.snow.css'  
import 'quill/dist/quill.bubble.css'

全局引用就丢在main.js，局部丢在相关vue文件内的 export default 上方

附上引入以及使用方法：

安装： 

npm install vue-quill-editor --save

1)在main.js中引入 

import vueQuillEditor from 'vue-quill-editor' // 引入富文本工具

2)在main.js中使用 

Vue.use(vueQuillEditor)

3) 页面上

<quill-editor ref="myTextEditor"  
 :content="content"//内容对应的字段  
 :options = "editorOption"  
 @change="onEditorChange($event)">//内容改变事件  
</quill-editor>

4) js 

export default {  
components: {  
quillEditor  
},data(){  
content:""  
}
